# Task Manager

A minimal, dark-themed task management application built with vanilla JavaScript.

## Features

- Add, complete, and delete tasks
- Filter tasks by status (All, Active, Completed)
- Persistent storage using localStorage
- Real-time statistics tracking
- Clean, professional UI design
- Responsive layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API

## How to Use

1. Type your task in the input field
2. Press Enter or click "Add Task"
3. Check the checkbox to mark tasks as complete
4. Use filters to view different task categories
5. Click "Clear Completed" to remove finished tasks
